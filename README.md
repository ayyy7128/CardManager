# 卡片管家 CardManager

卡片管家是一款本地优先的银行卡、账户、提醒任务和资产记录管理工具。它适合同时管理实体卡、虚拟卡、多币种账户和投资类提醒的用户：可以按分组整理卡片，记录卡面图和银行 Logo，设置还款、保活、投资等任务，并导出包含数据、图片和自定义字体的加密备份。

数据默认保存在本机，不做云同步，不上传个人信息。

当前版本：**1.20.0**（versionCode 24）<br>
数据库 schema：**v10**<br>
平台：Android，Kotlin + Jetpack Compose + Room SQLite

## 主要功能

- 卡片分组管理：新建、编辑、删除、折叠分组，支持分组上下排序，也可开启无分组模式直接浏览卡片。
- 卡片管理：支持实体卡、虚拟卡、纯账户，记录银行、卡种、卡组织、币种、尾号、有效期、状态、备注、卡面方向、卡面图和银行 Logo，并可用 NFC 辅助填写公开 EMV 信息。
- 卡片浏览：支持图册模式和列表模式，支持搜索银行、卡种、卡组织、币种、尾号、备注和卡类型。
- 筛选：支持按卡组织/纯账户、币种、状态筛选。
- 日历任务：支持按月查看任务日期，查看当天任务、定期任务和投资任务。
- 提醒任务：支持每月、每周、每季度、每 N 天、一次性任务，并支持关联卡片。
- 小金库：记录快速存取流水；新增投资任务，可记录定投、存款、底仓、资金调整、关联卡片和归档金额。
- 数据总览：展示卡片数量、状态分布、卡片构成、卡组织、币种、分组、任务频率和资产流水等本地统计，总览和图表都支持自定义显示与长按拖动排序。
- 设置：全屏设置页，进入时带滑动过渡，支持跟随系统/浅色/深色模式、自定义字体、`.cmbak` 备份导入导出、导入合并/覆盖、竖屏/横屏卡面每行数量、卡片分组显示模式和底部 Tab 自定义。

## 1.20.0 更新亮点

- 小金库新增投资任务，支持多阶段定投、单次投入、初始金额、资金调整、自动非交易日判断、顺延、归档冻结和日志覆盖。
- 日历页和小金库页的右下角加号都改为展开菜单，投资任务会进入独立全屏编辑页。
- 日历页任务区支持折叠，当天栏会显示命中的定期任务和投资任务。
- 旧投资类任务会自动迁移为新的投资任务，并同步原任务关联卡片。
- 快速记录会作为“小金库”里的独立项目展示，只包含手动添加的流水。
- 小金库总额支持在 CNY、HKD、USD 间切换，并按汇率换算显示项目余额。
- `.cmbak` 备份会包含投资任务数据和小金库币种设置，导入导出时可随其他本地数据一起迁移。
