# 更新日志

## 2.4.0 - 2026-07-15

本版本重做全屏页面与主导航，并新增外置卡片资源库，集中修复备份恢复、投资金额、交易日缓存和桌面小组件的数据可靠性问题。

- 卡片聚焦/编辑、投资详情/编辑和日历任务编辑改为 Activity 内真实全屏页面，移除最外层全屏 Dialog，并正确适配状态栏与底部手势区。
- 新增 Android 预测性返回手势支持；全屏页面会跟随手势进度移动并支持取消回弹，投资详情进入编辑后可按层级返回详情。
- 底部 Tab 移出 Scaffold 占位栏并覆盖悬浮在正文上方；宽度随可见 Tab 数量自适应，选中底色覆盖完整选项，并使用浅色/深色自适应半透明玻璃表面。
- 简化悬浮 Tab 材质：取消毛玻璃、透明背景、噪点和高光效果，改用稳定的实色主题背景，避免正文分界线透出形成横向白条；累计滚动足够距离后底栏才会收起或恢复，前三个页面的新增按钮会随底栏同步移动，并在底栏隐藏后停留于右下角。
- 新增标准 `.zip` 卡片资源包导入、覆盖更新、查看占用空间和删除功能，资源不打进 APK。
- 新增 119 条内置银行/发行机构目录和 66 个识别 Logo；银行改为全屏选择，支持中英文搜索、按首字母分组和右侧字母索引跳转。
- 银行选择页新增“找不到？”入口，可添加自定义银行名称和 Logo，或打开预填内容的 GitHub Issue 反馈缺少的银行；旧版手动填写的数据继续按自定义银行使用。
- 银行目录将“天星银行 / Airstar Bank”更新为“象象银行 / Elebank”，并保留旧名称兼容已有卡片和搜索。
- 新增 Fiat24、蚂蚁银行（香港）和蚂蚁银行（澳门），补充汇立银行与众安银行 Logo，替换完整的中信银行 Logo，并移除卡片编辑页 Logo 预览底色。
- 新增“中國建設銀行（亞洲）”和“中信銀行（國際）”，保留简体旧名用于搜索兼容并复用对应内地银行 Logo；汇丰各地区条目统一使用去底透明 Logo，Chase 与 Chase UK 统一图标。
- 象象银行（Elebank）新增净化放大后的透明 Logo。
- 新增 GitHub Releases 更新检查：每日最多自动检查一次，发现新版时展示说明并打开官方发布页；设置页支持手动检查，网络失败时启动保持静默。
- 压缩主页顶栏高度，并避免底部导航重复计算系统导航区，为正文释放更多显示空间。
- 卡片编辑页新增卡面资源库，支持搜索、储蓄卡/信用卡筛选和卡面预览；选中后自动填写银行、卡名、卡组织、卡类型、币种、卡面和资源包提供的银行 Logo。
- 从资源库使用的卡面会复制到卡片自有目录；银行目录已有对应 Logo 时复用内置资源，未收录 Logo 才复制资源包文件，因此删除资源包不会影响已有卡片。
- README 公开资源包的 ZIP 目录、JSON 清单、图片引用和校验限制；项目不提供第三方卡面下载脚本或生成后的卡面资源包。
- 覆盖导入改为先解密、限制解压大小、完整解析并验证备份，确认有效后再在单个 Room 事务中替换数据，避免损坏备份先清空现有数据或只导入一部分。
- 备份 JSON 使用系统 JSON 转义规则，避免备注等文本中的控制字符损坏备份。
- 新导出的 `.cmbak` 必须设置密码；旧版无密码备份保持导入兼容。
- 未来日期的初始资金和资金调整只在生效日期到达后计入投资项目余额。
- 小组件在任务、快速记录、投资项目和显示币种变化后主动刷新，并与 App 共用最近一次成功汇率缓存。
- 删除卡片或分组时，投资项目会保留并解除关联；删除确认文案同步说明实际行为。
- 当年及未来年份的交易日缓存按 7 天刷新；请求失败时优先使用已有旧缓存，并确保网络连接正确关闭。
- Room 开启 schema 导出，并新增投资未来日期计算单元测试。
- 版本升至 2.4.0（versionCode 32），数据库 schema 保持 v12。

## 2.2.0 - 2026-07-08

本版本新增 Android 桌面小组件，用于在桌面快速查看任务和小金库金额。

- 新增 2x2 和 4x2 两种小组件尺寸。
- 添加小组件时可选择显示具体日期和今日任务、小金库总额或单个投资项目金额。
- 今日任务会合并定期任务和当天投资任务，并只显示总项数；小金库金额使用当前小金库币种和本地兜底汇率显示，金额字号会按小组件宽度调整。
- 单个投资项目小组件会显示最近几期投资流水；配置页改为浅底高对比显示，4x2 布局改为更稳定的满宽结构。
- 小组件点击可直达对应页面：今日任务进入日历，小金库总额进入小金库，单个投资项目进入项目详情。
- 小组件配置按系统 widget id 保存在本机，不纳入 `.cmbak` 备份。
- 数据库 schema 保持 v12。
- 版本升至 2.2.0（versionCode 29）。

## 1.20.0 - 2026-06-20

本版本增强小金库记录能力，让日常流水、定投和存款计划可以在同一页面管理。

- 资产页改名为小金库，新增投资任务，可记录定投、存款、底仓和资金调整。
- 支持多阶段金额、单次投入、每 N 天/每周/每月/每日周期、自动非交易日判断、非交易日顺延和月末缺日期策略。
- 支持在投资任务详情中查看流水，并对定投/顺延/调整记录做覆盖或删除。
- 日历页和小金库页的右下角加号都改为展开菜单，投资任务进入全屏编辑页。
- 日历页去掉“全部任务”总栏，新增可折叠的“定期任务”和“投资任务”，当天栏会显示命中的投资任务。
- 小金库首页只展示投资项目，可在 CNY/HKD/USD 间切换总额和项目折算余额。
- 支持归档投资任务并冻结金额，可选择是否计入小金库总额。
- 旧投资类任务会自动迁移为新的投资任务，并同步原任务关联卡片。
- 快速存取流水会单独显示为“快速记录”项目，且只包含手动添加的流水。
- `.cmbak` 备份导入导出新增投资任务数据和小金库币种设置。

## 1.13.0 - 2026-06-13

- `.cmbak` 备份会保留任务的非交易日策略。

## 1.12.0 - 2026-06-13
